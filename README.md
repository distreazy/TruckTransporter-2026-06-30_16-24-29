Грузовик-Перевозчик (TruckTransporter)

Физический 2D-симулятор перевозки нестабильного груза по сложному рельефу.

## Описание

Игрок управляет тягачом с прицепом, перевозящим хрупкий груз. Задача — доехать от старта до финиша, сохранив как можно больше ящиков в кузове. Сложный рельеф, физика подвески и инерция груза делают поездку настоящим испытанием.

**Жанр:** 2D Physics Simulator / Arcade  
**Платформа:** PC (Windows)  
**Движок:** Unity 2D  
**Курсовой проект:** 2 курс, разработка игр

## Правила игры

### Цель
Довезти груз до финиша, сохранив максимальное количество ящиков в прицепе.

### Условия прохождения
- **Уровень 1:** 5 из 6 ящиков для разблокировки Уровня 2
- **Уровень 2:** 5 из 6 ящиков для разблокировки Уровня 3
- **Уровень 3:** 5 из 6 ящиков для завершения игры

### Ограничение по времени
- **Уровень 1:** На уровень даётся **30 секунд**
- **Уровень 2:** На уровень даётся **60 секунд**
- **Уровень 3:** На уровень даётся **75 секунд**
Если время истекает — поражение.

### Поражение
- Время вышло
- Потеряно более 1 груза

### Победа
- Довезён весь необходимый груз до финиша в пределах лимита времени

## Технические особенности

- **WheelJoint2D** для настройки подвески (Suspension, Damping)
- **HingeJoint2D** для сцепки тягача и прицепа
- **SliderJoint2D** для ограничения хода подвески
- **Динамическое смещение centerOfMass** для стабилизации
- **Physics Material 2D** с высоким трением для сцепления
- **PlayerPrefs** для сохранения прогресса
- **AudioSource** для звуков двигателя и событий
